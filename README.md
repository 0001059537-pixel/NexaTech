# NexaTech Landing Page

Site institucional para NexaTech — template de landing page com seções de serviços, sobre, FAQ e contato.

## Tecnologias

- Next.js (App Router)
- React + TypeScript
- Tailwind CSS
- pnpm

## Recursos adicionados nesta entrega

- Navegação com scroll suave entre seções
- Botão flutuante de WhatsApp para contato direto
- Feedback ao enviar o formulário (toast)

## Como rodar localmente

1. Instale dependências:

```bash
pnpm install
```

2. Rode em modo desenvolvimento:

```bash
pnpm dev
```

Abra http://localhost:3000

## Boas práticas Git

- Nunca commite diretamente na `main`.
- Crie uma branch para cada feature, por exemplo: `feat/layout`, `feat/whatsapp`, `fix/forms`.
- Faça commits pequenos e com mensagens claras.
- Abra Pull Requests para revisão e merge.
